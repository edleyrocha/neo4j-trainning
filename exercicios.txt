Exercício 8 – Liste todos os envolvidos no filme “Top Gun”, apresentando o nome e o cargo da pessoa. A lista deve estar ordenada primeiramente pelo cargo e depois pelo nome da pessoa. 

MATCH (pessoa:Person)-[r]->(filme:Movie) 
WHERE filme.title = "Top Gun"
RETURN pessoa.name as Nome, type(r) as Cargo
order by Cargo, Nome

Exercício 9 – O ator Barry Pepper, nascido em 1970, atuou como Dean Stanton no filme The Green Mile. Adicione essa informação ao banco. 

CREATE (:Person {name: "Barry Pepper", born: 1970 } )
  
MATCH (pessoa:Person {name: "Barry Pepper"}),
      (filme:Movie {title: "The Green Mile"})
CREATE (pessoa)-[:ACTED_IN {roles:["Dean Stanton"]}]->(filme)

Exercício  11 – Quantas e quais pessoas Wagner Moura deve conhecer para encontrar Demi Moore? 

MATCH p = shortestPath( (k)-[*]-(t) )
WHERE k.name = "Wagner Moura" AND t.name = "Demi Moore"
RETURN length(p), nodes(p)
